== Description ==

Use mobile to login your wordpress site by scanning QR Code.

**Main features / advantages**

* Simple and easy to use.
* No need to enter username or password while logging site.
* Compatible with all major web browsers: Firefox, Chrome, Internet Explorer, Safari, Opera.

== Installation ==

1. Download plugin file (.zip)
2. Extract zip file and upload folder /tagauth/ to /wp-content/plugins/
3. Go to "Plugins" and activate the plugin
4. Click on the menu "TagAuth Setting" in the dashboard.
5. Click on the menu "Get QR Key" to bind account with your mobile.
6. Use your mobile to login site by scanning QR Code.

== Frequently Asked Questions ==
 
= How do I change settings? =

From the Wordpress Administration go to "Plugins" and click on "Settings" under plugin name

= Is TagAuth using Javascript? =

Yes. Javascript is required for generating QR Code.
